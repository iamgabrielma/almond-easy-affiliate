== Description ==

Manage your affiliate links through shortcodes with this simple solution for WordPress. Keep track of your affiliate links and use them through your site and manage them in a simple database.

== Installation ==

=== Manual ===
1. Upload the Almond Easy Affiliate Data folder to the plugins directory in your WordPress installation
2. Activate the plugin.
3. Navigate to the "Settings" menu in the administrator dashboard and you will find a submenus called "Almond Easy Affiliate". You can also access to the configuration menu clicking on "Settings" on the plugin list once it has been activated.

That's it! Now you can easily fill the required fields "Text to display" and "Affiliate Link" and click on the Save button to save it in the database, once you refresh the window or click on Refresh you will be provided with the Shortcode that only needs to be copied and pasted anywhere in your WordPress site.
