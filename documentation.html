A) Installation - top

Once you have your zip file from Envato you only need to upload it through the administration area in your WordPress website. Click on Dashboard > Plugins > Add New.

In the Add Plugins page you only have to click on the "Upload Plugin" button and select the file containing your new plugin.

Once you click the Install now button WordPress will take care of the process and show you a new page similar to this one

The plugin will appear in your Installed Plugins list, and once you activate it you will see few options under its name :

  • Deactivate: You can deactivate or activate Almond Easy Affiliate clicking here.
  • Edit: This link will redirect you to the source code of the plugin. I do not recommended to change anything from the source code if you do not fully understand what are you doing. In any case I'm available for customizations, just contact me if you need to customize Almond Easy Affiliate to fit your needs.
  • Settings: Clicking here will redirect you to the Almond Easy Affiliate options page, where the plugin can be used and configured.
  • More plugins by AlmondWP: More plugins created by me :)

The next step will be clicking on Settings so you can use and configure the plugin in your website. You can also find the configuration options in the administrator dashboard under Settings > Almond Easy Affiliate.

B) Using Almond Easy Affiliate - top

  • Text to display: The title you'd like to show in the front of your website in the top of the widget, if any.
  • Affiliate link: a ticker is the one to four letter code that represents a particular stock or company in the stock market, writing the ticker of any company available will return the actual price and volume about the company.
  • Save: Save the current info written in the previous fields into the database, it also creates a shortcode so affiliate links can be easily used through the website. To check out the latests updates in the database, click on Refresh button.
  • Edit: Edit information already entered, once clicked it will appear a new field that must be filled, the ID of the entry to be edited.
  • Add Image: For adding a new image, once clicked the button, new fields and buttons will appear, you must fill the field with the local URL and save it into the database, the plugin will create a new shortcode for you so the affiliate image can easily added through the content on your site.
  • Refresh: Updates the latest information from the database.
    • Saving a link in the database will create an affiliate shortcode that can be copied and pasted anywhere in WordPress, that will generate the link through your content.

C) Configuration options - top

  • ID: Shows the unique ID of each affiliate, the shortcodes will work with this ID, either links or images attached.
  • Name: The text that will be displayed as a link.
  • Affiliate link: The affiliate link.
  • Shortcode: There are two types of shortcode, links or images, the plugin will provide the shortcode for links by default and will also provide the shortcode for images if an image has been attached.
  • Attached Image: If an image has been attached as affiliate, this will display the local URL where the image is saved. Only for information purposes.
  • Delete: Clicking on delete will delete the affiliate link from the database.

C) Frequent Asked Questions - top

Why cannot I add manually or change the ID number associated to each affiliate?

We decided to avoid that the user can do that for a simple reason, so the user don't have to keep track of outdated affiliates, each affiliate have an unique ID that will show the unique affiliate link or do not show anything in case that doesn't exist in the database. This way the user is sure that there is no outdated or wrong affiliate links through the website and edit them one by one.
